# Carpeta Denuncias - Estructura

```
denuncias/
├── datos/          # Archivos CSV de denuncias (2014-2025)
├── cuadros/        # Cuadros estadísticos Excel por año
├── archivos/       # Archivos RAR originales
└── README.md       # Este archivo
```

- **datos/**: Contiene los CSV con las denuncias educativas. El dashboard los carga automáticamente.
- **cuadros/**: Cuadros estadísticos en formato Excel (SUPEREDUC).
- **archivos/**: Archivos comprimidos RAR con los datos originales.
